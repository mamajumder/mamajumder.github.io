Structured vs Unstractured data

  • Structured
    • data table
    • example: excel, html table
    • tools: SQL, R, excel
  • Semi structured
    • xml, json
    • example: some specific web
    • tools: R package rvest, rjson
  • Unstructured
    • plain text
    • example: twitter, web sites
    • tools: R package stringr, twitteR

Gathering data from web

  • method 1: scrap raw text data
    • clean and analyse
  • method 2: xml or json data
  • method 3: use API
    • twitter, facebook, weather data

method 2: data as html or xml

  • Data can be hidden in html
    • html tag
    • tag attribute
    • html text
  • To glean data
    • locate the data by inspection
    • find if it is html tag or attribute or text
    • obtain the path using google chrome

understanding xml path

    • right click on the object that contains the data >> copy xpath

map <- location_map %>% 
  html_nodes(xpath = '//*[@id="section3"]/div/div[2]/div[6]/img') %>%

method 3: Scraping data using API

  • Some web applications provide API so that end users can access data they provide
  • We will show how to get data from tweeter using their API
  • Use R package twitteR to extract and work with twitter data
    • you can search specific tweets by person, location, topic etc.
    • analyse re-tweets, counts etc.
    • install twitteR from github not CRAN what you usually do
    • for details visit https://github.com/geoffjentry/twitteR/

Sources of tweets

r_tweets <- searchTwitter("#rstats", n=300)
sources <- sapply(r_tweets, function(x) x$getStatusSource())
sources <- gsub("</a>", "", sources)
sources <- strsplit(sources, ">")
sources <- sapply(sources, function(x) ifelse(length(x) > 1, x[2], x[1]))
source_table = table(sources)
df <- data.frame(names(source_table),source_table)
library(ggplot2)
ggplot(df[df$Freq>8,], aes(reorder(sources,Freq),Freq)) +
  geom_bar(stat="identity") + coord_flip()
